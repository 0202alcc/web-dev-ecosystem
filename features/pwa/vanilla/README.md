# Vanilla PWA

A minimal implementation of a Progressive Web App using vanilla JavaScript.

## Overview
This project demonstrates core PWA functionality without any frameworks, including:
- Service Worker for offline support
- Web App Manifest for installation
- Basic offline capabilities

## Getting Started

1. Serve the files with a static server.
2. Access via HTTPS for PWA features.
3. Install the app on supported devices.
