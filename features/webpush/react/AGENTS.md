# Agents Context - React WebPush

## Project Purpose
To demonstrate a production-ready Web Push implementation using Next.js and MagicBell.

## Key Technologies
- **Next.js**: Framework.
- **MagicBell**: Push provider.
- **Tailwind CSS**: Styling.

## Guidelines
- **Mobile First:** Ensure UI works perfectly on iOS PWA context.
- **Authentication:** Maintain HMAC security patterns.
